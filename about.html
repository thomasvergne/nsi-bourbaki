Conception du site

Ce site a été conçu en prévision de support des fonctionnalités de HTML5 et CSS3. Certaines APIs CSS modernes ont été utilisées comme flex ou background-gradient... Il se peut donc que certains effets visuels puissent être absents si le navigateur utilisé est ancien.

La conception a été effectuée avec l'aide du framework CSS Tailwind et d'un retravail, optimisation et formattage du code CSS généré afin d'obtenir un fichier d'output CSS le plus propre qu'il soit.
